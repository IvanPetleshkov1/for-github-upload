1.Какво е HTTP?
HTTP -Hypertext Transfer Protocol ,  интернет протокол за обмен на данни между два компютъра.

2.Какво е HTTPS?
HTTPS-Hypertext Transfer Protocol Secure протокол за ЗАЩИТЕНА комуникация в компютърна мрежа, широко разпространена в интернет.

3.Каква е разликата между HTTP и HTTPS?
HTTP -Данните се взимат и се пращат към сървър във вид на обикновен текст.
HTTPS-Данните се взимат във вид на обикновен текст, но преди да се пратят се кодират чрез SSL като едва след като са кодирани се пращат към получателя им.

4.Каква е разликата между GET и POST заявка?
GET заявки- данните се изпращат чрез URL адреса (виждат се в URL адреса).
POST заявки-данните вграждат в обекта на формата и се изпращат директно от вашия браузър до сървъра.(НЕ се виждат в URL адреса)

5.Какво са HTTP хедъри?
HTTP хедърите са полета с информация, в заглавната част на HTTP заявката и HTTP отговора, предавани между клиент и сървър.

6.Възможно ли е с JavaScript да достъпваме бисквитките в сайта и как?
Да,възможно е като използваме document.cookie

7.Възможно ли е да запишем функция в променлива?
Да!

8.Каква е разликата между == и === ?
== -сравнява променливите по стойност
===-сравнява променливите както по стойност така  и по тип

9.Какво е JSON и какво е AJAX?
JSON-използва JavaScript синтаксис, но JSON форматът е само чист текст. Форматът на JSON е синтактично идентичен с кода за създаване на JavaScript обекти.Заради тази прилика JavaScript програма може лесно да конвертира данни от JSON в JavaScript обекти. 
AJAX-асинхронeн JavaScript, нова техника за създаване на по-добри, по-бързи и по-интерактивни уеб приложения с помощта на HTML, CSS и Java Script.

10.Какво е async и await?
Async-дефинира асинхронна функция, която трябва да изпълнява една или повече задачи.
Await-Това е оператор , койт се поставя преди извикването на асинхронна операция, която представлява обект Promisе.

11.Какво е глобална променлива и какво е локална променлива?
Глобална променлива-декларирана е извън всички функции,съществува до края на изпълнението на програмата и може да се използва във всички функции без да има нужда да се декларира всеки път в отделните функции.
  Локална променлива-декларирана е в съответната функция , в която ще се ползва.Използва се само и единствено в съответната функция, в която е декларирана.

12.Как може да се използва ключовата дума “this”?
Ключовата дума "this" се отнася до текущия обект, в който се пише кодът.
